# ClickShow
用于提示鼠标点击、鼠标位置。

**特色功能**
- 鼠标点击时显示波纹特效，每个按键对应不同的颜色；
- 支持跟随鼠标的位置指示圆标；
- 支持多屏DPI感知；
- 支持开机自启动；
- 绿色单文件；

# 使用
- 启动后自动启动点击提示并缩小到系统托盘。
- 点击托盘图标打开主窗口。
- 如需在任务管理器、以管理员身份启动的窗口上使用，请将程序(1.3.1+版本)放到C:\Windows或C:\Program Files目录下。这时候程序会自动提升权限。

## 可能会遇到的问题
- 特效丢失：鼠标挂钩丢失了，这时候需要重启一下程序。
- 自启动不生效：被各类管家或启动软件拦截，请在这些软件里设置。
